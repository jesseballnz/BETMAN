# BETMAN Mobile

BETMAN Mobile is the mobile execution companion for the BETMAN platform.

It is designed for continuity under live conditions: fast context recovery, BETMAN Pulse monitoring, tracked-runner management, and race reopen workflows away from the desktop workspace.

## Stack
- Expo
- React Native
- React Navigation
- Expo Haptics
- Expo Notifications
- AsyncStorage

## Product role
BETMAN Mobile is built for:
- BETMAN Pulse monitoring
- tracked-runner follow-up
- race reopen workflows
- mobile continuity away from the desktop workspace

It is not a stripped-down viewer.
It is an operator-grade mobile surface for live race-day work.

## What has improved
- stronger Pulse focus controls
- clearer tracked lifecycle views
- better market movers filtering
- safer cross-surface race reopen behavior
- improved menu consistency and narrow-screen usability

## Positioning
BETMAN Mobile keeps the operator connected to the same decision loop as the web workspace:
- see signal
- reopen race
- review context
- track the runner
- monitor outcome
